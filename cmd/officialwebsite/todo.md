# 核心任务
通过比对Apk长度判断是否为最新版本，以此来更新Apk文件信息

## 任务流程
1. 从redis中获取官司及apk相关信息比如官网链接、Apk长度，版本
2. 查找到apk链接 通过playwright模拟
   2.1 通过打开官网，查找下载标识 比如Android下载 Android版下载，等这个关键词可以配置
   2.2 模拟点击， 获取下载链接
   2.3 若获取到链接 获取长度
3. 比对Apk长度，如果一致则更新Apk文件信息，否则不更新
4. 不一致刚下载apk.
5. sftp传输到对应服务器
6. 获取apk信息更新到一个rest api接口

## 施行编码
core/queue目录需要一个redis类基础
services目录需要redis类一个读取队列，交给浏览器模拟类。一个下载服务已存在
